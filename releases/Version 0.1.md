### 🚀 Release 0.1 - Foundation

Cette première version pose les bases du template avec une stack moderne et optimisée pour le développement.

**Points clés :**
* **Framework :** Symfony 8.0 & PHP 8.4.
* **Stack Technique :** PostgreSQL 18 et Mailpit via Docker Compose.
* **Frontend :** Intégration de Tailwind CSS avec Symfony UX (Stimulus & AssetMapper).
* **DX (Developer Experience) :** Automatisation via Symfony CLI (watch Tailwind et services Docker).
* **Documentation :** README à jour avec instructions d'installation et workflow.
