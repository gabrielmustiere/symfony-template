### 🚀 Release 0.2 - Quality & UI Base

Cette version se concentre sur l'amélioration de la qualité du code (QA) et la mise en place d'une structure d'interface
utilisateur (UI) solide.

**Nouveautés et modifications :**

* **Qualité de code (QA) :**
    * Intégration et configuration de **PHPStan** (niveau élevé) pour l'analyse statique.
    * Ajout de **PHP-CS-Fixer** pour garantir un style de code cohérent et standardisé.
* **Interface Utilisateur (UI) :**
    * Création d'un **template de base (`base.html.twig`) enrichi** : ajout d'une mise en page avec sidebar responsive,
      header de navigation et zone de contenu principale.
    * Intégration de **Tailwind Elements** pour des composants UI avancés.
* **Documentation :**
    * Mise à jour de la roadmap dans le `README.md`.
    * Documentation des nouveaux outils de développement.
* **Maintenance :**
    * Optimisation du fichier `.gitignore`.
    * Mise à jour des dépendances de développement.
